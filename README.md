# Actividad_Arboles_estructuras
Actividad de clase
Creacion de archivo con no,bre, id y promedio del estudiante e implementacion de listas y arboles ABB y B+
El repositorio cuenta con comentarios que explican las lines ade codigo mas relevantes para que sea mas disiente para el epectador y el desarrollador mismo
Se puede evidenciar los tiempos de ejecucion de busqueda de cada sistema de un ID unico y una comparativa entre estos
Actividad realizada con base en los aportes de ChatGPT Y Gemini
